{% header h2 | Why would you want to use markdown links? %}

I don't know about you, but I love things that look nice. To me, a message with a couple links (especially if they are long) doesn't look nice.

With markdown links, you can also add links to a channel's topic.

{% header h2 | Using markdown links %}

If you send a message (or edit a channel's topic) as [title](https://link.com), the message will be just "title" hyperlinked.
