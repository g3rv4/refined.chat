{% header h2 | How can install this extension on Opera? %}

You can install Chrome's extensions on Opera! In order to do so you should:

  • Install the Install Chrome Extensions extension
  • Visit the extension's page on the Chrome store
  • Click on the "Add to Opera" button
{% header h2 | Why isn't this extension on the Opera store? %}

It used to be there... but their review times are crazy. After waiting for 2 months for a review, I decided to remove it from the Opera store since Chrome extensions work just fine on Opera.
