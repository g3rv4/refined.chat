--- layout: default title: Taut - a tool for Slack description: Taut is a small extension to personalize Slack on your browser. ---

Here's a quick list of all the features. Follow the links for details on what they do :) Originally, I recorded a video showing the extension, but keeping it updated was really challenging.

{% header h2 | Mute users %}

Do you have some noisy bots you'd like to mute? Taut lets you do that.

{% header h2 | Use markdown links %}

You can write things like [Check this amazing extension](https://taut.rocks) and that's going to appear as Check this amazing extension for you and for everybody on the channel (even if they are not using Taut). Want to see an example? Check it out!

{% header h2 | Generate hangout urls %}

Want to invite somebody to a hangout? I find that generating a link is easier and faster than the alternatives.

{% header h2 | Unread count on the title %}

Sometimes (well, all the time) I'm on a tab that is not Slack. When that happens all I see is an asterisk on the title when there are some unread messages. Taut can add a count of unread messages on the page title.

{% header h2 | Move reactions to the right %}

Reactions are an amazing way of providing feedback without writing... but they can take lots of vertical space. If you move them to the right they no longer do!

{% header h2 | Threads on channel by default %}

Do you wish replies defaulted to "Send to #channel"? Taut can do that for you.

{% header h2 | Make "send to channel" sticky %}

Once you send a reply to the channel, by default Slack unticks the checkbox for your next message. Taut can keep your last message's selection if you want it to.

{% header h2 | Disable Google Drive previews %}

Google Drive previews are great, but they take lots of vertical space. Taut can replace these previews with the document url, so that you can click on it if you wish.

{% header h2 | Disable url previews %}

Url previews... the times they provide any useful information for me are extremely rare. That's why Taut can totally disable them.

{% header h2 | Hide the status emojis %}

And last but not least, in order to decrease the noisiness of your workspace, Taut can hide everyone's status emojis.
